# CoffeeBytes
Coffee Bytes 1.0.6

## Works with Eclipse
ver 4.19.0

## Insructions

1. Copy **features** and **plugins** to the Eclipse installation
2. Close and reopen Eclipse
3. Go to Windows > Preferences > Java > Editor > Folding
   1. Choose **Coffee Bytes Java Folding** from drop down
